# Shell & Tube Heat Exchanger Design Datasheet (HTML)

## Overview

This project is an HTML-based engineering datasheet developed for the thermal and hydraulic design of a **Shell & Tube Heat Exchanger (STHE)**. The datasheet replicates the layout and calculations of a Microsoft Excel workbook using HTML tables and CSS, making it suitable for viewing in a web browser, printing, and future integration with web applications.

The design sheet includes equipment information, process data, thermodynamic calculations, sizing calculations, material selection, heat transfer analysis, pressure drop calculations, and overall heat transfer coefficient calculations.

---

## Features

* Excel-like table layout using HTML and CSS
* Professional engineering datasheet formatting
* Merged cells using `rowspan` and `colspan`
* Responsive table structure
* Print-friendly design
* Easy to modify and extend
* Structured sections for process and mechanical calculations
